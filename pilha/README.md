##### Uma pilha é um conjunto ordenado de itens no qual novo sitens podem ser inseridos e a partir do qual podem ser eliminados.   

* Não é possível ter uma visão lateral (de perfil) da pilha, só vemos o item que está no topo.   
* O único modo de verificar se duas pilhas são iguais é desempilhando cada uma delas e comparando.   
* Se for nescessário um registro dos itens intermediários que passaram pela pilha, esse registro deverá ser mantido em outro lugar, ele não existe dentro da prória pilha.   

> incluir == empilhar   
> remover == desempilhar
